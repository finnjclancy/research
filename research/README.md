# Equity Research ‚Äî Working Notes

A personal equity-research workspace. The aim: **anticipate when the market's perception of a stock's valuation will shift, and estimate a range for the move.** See `00_system/MANDATE.md` for the approach.

## Structure
- `00_system/` ‚Äî the operating system
  - `MANDATE.md` ‚Äî the research approach.
  - `RESEARCH_LOG.md` ‚Äî running changelog of what's been added.
  - `COVERAGE_LIST.md` ‚Äî the watchlist: 14 EU + 2 US names + 3 macro briefs.
  - `MENTAL_MODEL_FRAMEWORK.md` ‚Äî **the master checklist**: World ‚Üí Sector ‚Üí Company ‚Üí News-to-Range reflex (the *what to know*).
  - `NARRATIVE_ENGINE.md` ‚Äî **how perception becomes price**: the narrative life-cycle, the game theory of every market participant, and the `stock ‚Äî range ‚Äî reason ‚Äî follow-up` output protocol (the *how to think*).
- `01_macro/` ‚Äî Layer 1: world/regime view.
  - `ECB_RATE_REGIME.md` ‚Äî the current regime (2026 oil shock ‚Üí ECB's first hike since 2023).
  - `AI_STACK_WARS.md` ‚Äî the AI-complex narratives + game theory and the read-through to the watchlist.
  - `MACRO_SCENARIO_TREE.md` ‚Äî **the decision tree**: macro narrative-shifts ‚Üí sector perception paths ‚Üí company perception paths, with a live "call sheet."
- `02_sectors/` ‚Äî Layer 2.
  - `SECTOR_NARRATIVES.md` ‚Äî each industry's place in the world (why it exists, why invest, bull/bear, what shift it's levered to).
- `03_companies/` ‚Äî Layer 3: one factual file per name (all 16; template in the framework ¬ß3, sensitivity map in each ¬ß7).
- `05_narratives/` ‚Äî Layer 3 narrative maps: per-name percolating stories, bull/bear narratives, shift tells, scenario read-through.
- `06_history/` ‚Äî 10-year trajectory + latest earnings + sell-side/buy-side sentiment per name.
- `04_recaps/` ‚Äî **daily process audit**. One file per day (`YYYY-MM-DD.md` from `_TEMPLATE.md`); the calibration loop that sharpens the reflex.
- `INDEX.md` ‚Äî the master map: every name linked to its docs and the relevant mental models.

## How to read it (the narrative stack)
1. `00_system/NARRATIVE_ENGINE.md` ‚Äî the philosophy + psychology + game theory + output protocol.
2. `02_sectors/SECTOR_NARRATIVES.md` ‚Äî where each stock's industry sits in the world's story.
3. `01_macro/MACRO_SCENARIO_TREE.md` ‚Äî the macro‚Üísector‚Üícompany branches for picking what you like/dislike.
4. `03_companies/*.md` + `05_narratives/*.md` + `06_history/*.md` ‚Äî the per-name detail, narrative map, and trajectory.

## Workflow
Hold the regime + scenario tree in mind ‚Üí when news breaks, recognise *which branch just got more probable* ‚Üí read off the most-relevant name + direction ‚Üí layer the company's ¬ß7 elasticity + life-cycle stage ‚Üí deliver **"stock ‚Äî range ‚Äî reason ‚Äî follow-up"** ‚Üí log it in `04_recaps/` (calibration).
