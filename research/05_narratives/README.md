# Narrative Maps — the percolating stories per name

Narrative-first companions to the factual files in [`../03_companies/`](../03_companies/). Each map covers: the **sector's narrative weather**, where the stock **sits**, the **cross-trend web**, the **bull narratives that re-rate it up**, the **bear narratives that re-rate it down**, the **narrative-shift tells**, and the **scenario read-through** (to [`../01_macro/MACRO_SCENARIO_TREE.md`](../01_macro/MACRO_SCENARIO_TREE.md)).

**Use:** when picking names you like/dislike, read the map to know *what story you're betting on*, *what flips it*, and *which scenario branch it wins/loses in*. The whole point is to know the **market's perception** so you can price a news-driven re-rate fast.

## ★ Cross-cutting narrative clusters (the relative-value / pairs view)

The book sorts into a few master narratives. **Names sharing a narrative move together; names on opposite sides of one are natural pairs.**

### Cluster A — The AI build-out (the book's biggest shared exposure)
- **AI supply chain (demand = hyperscaler capex):** ASML, AMD, ABB (equipment), + Amazon (spender).
- **AI power demand (the electricity leg):** Iberdrola, Enel, ABB.
- **AI seller / arms-dealer of inference:** Amazon (AWS).
- **The shared trigger:** the next **hyperscaler capex guide (Amazon first)** — a cut hits *all* of ASML/AMD/ABB/utilities (S3); a raise extends them (S4). See [`../01_macro/AI_STACK_WARS.md`](../01_macro/AI_STACK_WARS.md).

### Cluster B — The rate axis (the mirror trade)
- **Helped by higher-for-longer (oil-shock hikes):** BNP, Crédit Agricole, Banca Mediolanum (NII).
- **Hurt by higher-for-longer:** Enel, Iberdrola (bond proxies), + AI-growth multiples (ASML/AMD/Amazon/Zalando).
- **The pair:** long banks / short utilities is the *hawkish* expression; reverse it for the *dovish pivot* (S1, if oil rolls over).

### Cluster C — China
- **China property:** Kone (cleanest proxy).
- **China consumer:** Carlsberg (beer), Unilever (EM).
- **China chips / localization:** ASML (DUV/TAM), Melexis (China-Speed localization — the one name where China is a *growth* story).
- **China stimulus** is the shared upside catalyst (Kone modernisation, Carlsberg consumer).

### Cluster D — GLP-1 / obesity
- **The drug-maker:** Novo Nordisk (losing to Lilly).
- **The demand-destruction targets:** Unilever (exiting Foods → McCormick to de-risk), Carlsberg (reports "no significant impact yet").

### Cluster E — Idiosyncratic / low-macro-correlation (diversifiers)
- **Novo Nordisk** (clinical/competitive newsflow), **Embracer** (corporate-action/IP) — both trade on their *own* stories more than rates/oil/AI. Useful book diversifiers.

### Cluster F — Reshaping / self-help (corporate-action value)
- Unilever (Ice Cream demerged, Foods → McCormick), Carlsberg (Britvic), BNP/ACA (AXA IM/Amundi fee pivot), Embracer (breakup), ABB (sold Robotics), Iberdrola (Avangrid 100%). *Each has a structural re-rating lever independent of the macro.*
